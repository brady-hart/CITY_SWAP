A geography game with a number puzzle twist!
1. Type city names. Type as many city names from memory as you can! Large pins represent the seven largest populations in a region.
2. Swap for the right color. Swap pins by number (a 6 for another 6... a 3 for a 3 and so on). Use items like the Inspector and the Wild pin.
3. Clear 5 regions to win. Clear a region by swapping its pins around the map until they're all the same color. Then clear 5 regions to win!
